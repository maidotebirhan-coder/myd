Personal Website — Template

This is a small personal-website scaffold built with HTML, CSS, and JavaScript. Use it as a starting point for the Web Programming Assignment.

Structure
- index.html — Home page
- about.html — About page
- education.html — Education background
- skills.html — Skills
- projects.html — Projects
- contact.html — Contact (demo form)
- css/styles.css — Styles
- js/main.js — Small scripts
- images/ — Profile and project images

How to use
1. Customize the HTML files (replace "Your Name", text, and images).
2. Replace placeholder images in `images/` with your screenshots and profile photo.
3. Test locally by opening `index.html` in a browser.

Host on GitHub Pages
1. Create a new GitHub repository and push this project.
2. In the repository Settings -> Pages, select branch `main` (or `master`) and root folder `/`.
3. After a minute the site will be available at `https://<your-username>.github.io/<repository>/`.

Submission
- Provide the GitHub repo link.
- Provide the GitHub Pages URL.
- Attach a ZIP of the project source.

Create a ZIP on Windows (PowerShell)
- From the project parent folder run:

Compress-Archive -Path .\* -DestinationPath ..\yourname-portfolio.zip -Force
